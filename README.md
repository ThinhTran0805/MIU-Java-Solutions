# MIU Entry Exam Solutions 🎓

A collection of Java solutions for the programming entry exam at **Maharishi International University (MIU)**.

---

## 📂 Project Structure
The project follows the standard Maven/Gradle layout for seamless integration and scalability:

- `src/main/java/arrays/`: Array manipulation and logic-based algorithms.
- `src/main/java/strings/`: String processing and character manipulation challenges.
- `src/main/java/numbers/`: Mathematical problems, number sequences, and logic.

---

## ✅ Solved Problems

### 1. Arrays
* **[Zero-Plentiful](src/main/java/arrays/ZeroPlentiful.java)**: Evaluates if an array contains at least four 0s and ensures every sequence of consecutive 0s has a minimum length of 4.

## 📝 How to Run
Each solution is a self-contained Java class with a `main` method including various test cases. 
1. Clone the repository.
2. Open the project in **IntelliJ IDEA**.
3. Navigate to the desired file in `src/main/java/`.
4. Right-click and select **Run 'ClassName.main()'**.
